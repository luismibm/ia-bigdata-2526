# Análisis de sentimiento

El análisis de sentimiento es una técnica de procesamiento del lenguaje natural que utiliza la lingüística computacional y el machine learning para detectar el tono emocional detrás de los datos de texto.

Mientras que el análisis de sentimiento se centra en captar la emoción y la opinión dentro del texto, el procesamiento del lenguaje natural es la tecnología general que da a las máquinas la capacidad de trabajar con el lenguaje humano.

Es un caso de uso específico dentro de la disciplina más amplia del machine learning, generalmente utilizando modelos de machine learning supervisados.

Un ejemplo es Elastic.


### ¿Cómo funcionan?

1. **Ingesta de texto:** Se recopilan datos de texto sin procesar de diversas fuentes como correos electrónicos, tickets de soporte, logs de chat, redes sociales y reseñas de clientes. Estos datos se ingresan a un pipeline de procesamiento.

2. **Preprocesamiento de texto:** El texto se limpia y normaliza para facilitar el análisis.

3. **Extracción de características:** Se crean representaciones numéricas del texto para que los modelos puedan procesarlo.

4. **Clasificación de sentimientos:** El texto se clasifica en categorías de sentimiento (positivo, negativo, neutral).

5. **Salida y puntuación:** Cada texto recibe una puntuación de sentimiento en un continuo (por ejemplo, de -1 a +1) o una etiqueta categórica.

6. **Comentarios y actualizaciones:** El modelo puede ajustarse o reentrenarse con datos etiquetados para mejorar su precisión.
